# Issue 06: ProjectTimelineEditor (Refatoração Admin)
**Data e Hora de Geração:** 11/05/2026 às 11:54 (Horário de Brasília)

## Descrição
Refatorar a forma como as subtarefas do projeto são gerenciadas no painel administrativo para permitir visão global da timeline.

## Requisitos
- **Refatoração**: Alterar o `ProjectForm` no Admin.
- **Interface**: Implementar um sistema de Accordion (Acordeão) para as 5 etapas padrão.
- **Behavior**: O admin deve conseguir adicionar/remover subtarefas de qualquer etapa (Estratégia, Design, etc.) sem precisar que aquela etapa seja a "etapa atual" do projeto.

## Comportamento Esperado
Facilitar o planejamento completo do projeto logo na criação, permitindo definir todas as entregas de uma só vez.
