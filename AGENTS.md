# Project Focus and Constraints

## Focus
This project is a high-end web agency platform (DIFFUSE). The primary focus is on modern, high-performance, and visually distinctive web applications.

## Exclusions
- **Audio Systems:** This system does NOT include audio processing, playback, or engineering features. All previous audio-related prototypes (like Audio Lab) have been removed to keep the core focus on web architecture and design.
- **Sound Notifications:** User notifications are strictly visual. No audio feedback systems are implemented.
